# suwappu-perps-trader

Perpetual futures trading with [Suwappu](https://suwappu.bot) + HyperLiquid.

```bash
# TypeScript
npm install && export SUWAPPU_API_KEY=suwappu_sk_... && npx tsx trader.ts

# Python
pip install suwappu && export SUWAPPU_API_KEY=suwappu_sk_... && python trader.py
```

10+ perp markets | Up to 40x leverage | Live PnL tracking
